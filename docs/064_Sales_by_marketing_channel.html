Felipe Leandro Aguazaco

2022-01-06

1 Problem

Given the following data set, can you plot a chart that shows the percent of revenue by marketing source? You can assume that purchase_value is revenue in this case, and you’ll have to calculate the percent of revenue by source prior to plotting.

4 Resolving in R

div(
  ggplotly(
    data %>% 
    group_by(source) %>% 
    summarize(revenue = sum(purchase_value)) %>% 
    mutate(rev_percentage = round(revenue / sum(revenue) * 100, 
                                  digits = 2)) %>% 
    mutate(source = reorder(x = source, 
                            X = rev_percentage, 
                            FUN = min)) %>%
    ungroup() %>% 
    ggplot(mapping = aes(x = source, 
                         y = rev_percentage)) + 
      geom_col(width = 0.65, 
               colour = "black", 
               fill =  viridis::viridis(n = 3, 
                                        option = "E")) +
      labs(title = "Percent of Revenue by Marketing Source", 
           x = "Marketing Source", 
           y = "Percent of Revenue") + 
      theme_bw() +
      theme(plot.title = element_text(hjust = 0.5))
  ), 
align = "center")

import pandas as pd

# Problem
data_agg = data_python \
  .groupby(by = 'source', 
           as_index = False, 
           sort = False) \
  .agg(total_revenue = ('purchase_value', 'sum'))

data_agg['percent_revenue'] = round(data_agg.total_revenue / sum(data_agg.total_revenue) * 100, 2)
